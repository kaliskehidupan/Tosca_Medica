**ToscaMedika** adalah platform manajemen rekam medis berbasis web yang dirancang untuk mendigitalisasi administrasi klinik dan riwayat kesehatan pasien. Sistem ini dikembangkan menggunakan **Laravel 12** dan **Bootstrap 5** dengan fokus pada antarmuka yang bersih (*Tosca Theme*) dan pengalaman pengguna yang intuitif.

---

## âœ¨ Fitur Utama

* **Role-Based Access Control**: Pemisahan dashboard yang aman antara **Superadmin** (Manajemen User & Akun Petugas) dan **Petugas** (Operasional Medis).
* **Manajemen Data Medis**: Pengelolaan database Pasien, Dokter, dan Inventaris Obat secara terintegrasi.
* **Rekam Medis Terintegrasi**: Pencatatan riwayat pemeriksaan (keluhan, diagnosa, tindakan) dengan relasi obat *Many-to-Many*.
* **Dashboard Analitik**: Visualisasi data real-time menggunakan **Chart.js** untuk memantau tren kesehatan pasien dan ketersediaan stok obat.
